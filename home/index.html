📦 安装指南

三步完成安装和配置，立刻开始使用

1

下载插件文件

点击下方按钮下载插件压缩包，解压到本地文件夹

2

加载到 Chrome

打开 chrome://extensions/，开启「开发者模式」，点击「加载已解压的扩展程序」，选择解压后的文件夹

3

配置 API Key

点击工具栏 🦊 图标 → 设置 → 选择模型 → 填入 API Key → 保存。也可接入自建服务，数据完全私有

❓ 常见问题

需要付费吗？

插件本身免费可用。AI 功能需要各模型提供商的 API Key，按使用量计费。日常轻度使用成本极低，通义千问和智谱均有大量赠送额度。

支持哪些浏览器？

支持所有 Chromium 内核浏览器：Chrome（119+）、Edge、Arc、Brave、Opera 等。要求浏览器支持 Manifest V3 和 Side Panel API。

翻译质量怎么样？

翻译质量取决于你选择的 AI 模型。推荐使用 DeepSeek 或通义千问，翻译效果流畅自然，比传统机器翻译更懂上下文。

安全吗？我的 API Key 会泄露吗？

安全。API Key 只存储在浏览器本地 (chrome.storage.local)，仅用于向对应 API 发送请求。如需完全隐私，可使用自己的模型服务，所有数据不经过第三方。

为什么不用 Chrome 自带翻译？

Chrome 内置翻译依赖 Google 服务，国内用户无法使用。小叶助手通过国内 AI 模型提供翻译，无需梯子，效果更好。

怎么更新插件？

关注 GitHub 仓库更新，下载最新版本后在 chrome://extensions/ 重新加载即可。我们会持续迭代功能。
